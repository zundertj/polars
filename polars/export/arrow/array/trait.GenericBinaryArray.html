Expand description

Trait that BinaryArray and Utf8Array implement for the purposes of DRY.

Safety

The implementer must ensure that

  1. offsets.len() > 0
  2. offsets[i] >= offsets[i-1] for all i
  3. offsets[i] < values.len() for all i
