Enum simd_json::value::owned::Value

source · 
pub enum Value {
    Static(StaticNode),
    String(String),
    Array(Vec<Value>),
    Object(Box<Object>),
}
Expand description

Owned JSON-DOM Value, consider using the ValueTrait to access it’s content. This is slower then the BorrowedValue as a tradeoff for getting rid of lifetimes.
